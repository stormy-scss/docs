# Rounding

Applies a `border-radius` based on `rem` units

## Mixins
### Main

Apply on all sides

- `rounded-sm`
- `rounded`
- `rounded-md`
- `rounded-lg`
- `rounded-xl`
- `rounded-2xl`
- `rounded-3xl`
- `rounded-full` (Circular shape, used commonly for avatar images)
- `rounded-none`

### Sides

Apply on specified side

- `rounded-t-none`
- `rounded-r-none`
- `rounded-b-none`
- `rounded-l-none`
